Returns

Remarks

  • エディタでは常に推奨解像度を取得できません

resolution int2

Returns

Remarks

  • カメラごとにこのAPIを呼び出すことで、それらを同時に更新することが可能です
  • ただし、同じ名称のカメラは、同時に更新することができません
  • テクスチャーが未生成だった場合は、内部で勝手に適当な解像度で生成します
